# State and Local Integration

## Tiered Participation Model

---

## Participation Tiers

### Tier A: Full Participation
**For:** Large agencies with dedicated narcotics units
- Full API connection to unified data lake
- Real-time data contribution
- Access to Tier 1 and Tier 2 data
- Participation in SURGE OPS planning and execution
- Direct CASE BUILDER output access

### Tier B: Standard Participation
**For:** Mid-size agencies
- ODMAP/NARC data contribution
- Standardized OD intake forms for dealer identification
- Access to Tier 1 data
- Receive NARC CRI alerts for jurisdiction

### Tier C: Basic Participation
**For:** Small and rural agencies
- ODMAP contribution via mobile app
- Access to Tier 1 alerts for jurisdiction
- No API infrastructure required

## State-Level Coordination

Each state designates a Single Point of Contact (SPOC), typically:
- State HIDTA coordinator, or
- State police intelligence center, or
- State fusion center

The SPOC manages access for all state and local agencies within the state.

## The On-Ramp

No agency is too small to contribute. A two-officer rural PD responding to a fatal OD enters the event in ODMAP and preserves the victim's phone. That single action contributes targeting intelligence to the national system.

A county deputy in rural New Hampshire generates the same quality of actionable intelligence as a DEA special agent in Los Angeles.

## Funding Lever

Access to counter-fentanyl appropriation funds is contingent on:
1. Connection to the unified data lake via API (Tier A) or ODMAP contribution (Tier B/C)
2. Compliance with standardized data formatting
3. Participation in de-confliction protocols
4. Contribution of overdose event data
5. Submission of standardized victim-phone intelligence intake forms for OD deaths
6. Acceptance of credit-sharing protocols

Agencies that do not participate do not receive dedicated counter-fentanyl funds.
