# Funding Analysis

## Cost, Existing Appropriations, and Return on Investment

---

## Annual Cost at Full Operating Capability

| Component | Annual Cost |
|---|---|
| NFOC Operations (staff, facilities, overhead) | $20-30M |
| NARC (CRI engine, alerting, visualization) | $5-8M |
| Kill Chain modules (Precursor Watch, Chain Trace, Trade Trap, Bank Watch) | $15-30M |
| Dealer Down modules (Social Scan, Dark Watch, Payment Trace, OD Map, Identity Fusion, Case Builder) | $45-73M |
| Unified Data Lake (infrastructure, security) | $10-15M |
| API Integration and Platform Development | $8-12M |
| Training and State/Local Onboarding | $5-8M |
| **Total** | **$108-176M** |

## Existing Funding Sources (No New Legislation Required)

| Source | Amount | Authority |
|---|---|---|
| P.L. 119-75, Section 7036 (FY2026 counter-fentanyl) | $150M minimum | Appropriated |
| HIDTA Program (FY2025 budget) | $290M | Appropriated |
| DEA budget reallocation | ~$50M (1.5% of $3.4B) | Administrative |
| Forfeiture funds from fentanyl seizures | ~$200M+ annually | Existing authority |
| **Total available** | **$690M+** | |

The system is fully fundable from existing appropriations. Total available funding exceeds system cost by 4-6x.

## Return on Investment

| Metric | Value |
|---|---|
| Annual fentanyl deaths | ~70,000 |
| Economic cost per death | $4-11M (CDC estimate) |
| Annual economic impact | $280B-$770B |
| System annual cost | $108-176M |
| Deaths prevented to break even | 16-44 (out of 70,000) |
| 1% reduction in deaths (700 lives) | $2.8B-$7.7B in savings |

The system pays for itself by preventing 16-44 deaths out of 70,000. Any success beyond that threshold is pure return on investment.

## Context

- DEA annual budget: ~$3.4 billion
- System cost as % of DEA budget: 3.2-5.2%
- System cost as % of dedicated counter-fentanyl appropriation: 72-117% (supplemented by HIDTA and forfeiture)
- Estimated annual wholesale cartel drug revenue: $13.6-$49.4 billion
- System cost as % of cartel revenue: 0.2-1.3%
